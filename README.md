# Burger Shop Website
Modern and responsive Burger Shop website developed using HTML, CSS, and JavaScript.
## Features
- Responsive Design
- Modern Navigation Bar
- Products Gallery
- Product Details Modal
- Add to Cart
- Toast Notifications
- Cart Page
- Checkout Page
- Search
- Clean and User-friendly Interface
## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
## Pages
- Home Page
- Login Page
- Sign up Page
- Cart Page
- Checkout Page
